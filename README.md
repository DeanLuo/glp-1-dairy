# GLP-1 用药助手

帮助 GLP-1 使用者记录用药情况、体重变化、副作用追踪的微信小程序。

## 功能特性

- **用药记录** - 记录注射时间、剂量、注射部位
- **体重追踪** - 记录体重和腰围变化趋势
- **副作用记录** - 记录用药后副作用反应
- **用药提醒** - 设置定时提醒，不再错过用药时间
- **数据导出** - 导出历史记录，方便与医生分享

## 技术栈

- 微信小程序
- 微信云开发（数据库 + 云函数）
- Vant Weapp（UI 组件库）

## 开发指南

1. 克隆项目
2. 安装依赖：`npm install`
3. 导入到微信开发者工具
4. 配置云环境 ID
5. 开始开发
